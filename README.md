# React Boilerplate

This is a React boilerplate project that implements a dead simple 'Hello, world' example page and various configurations.

It comes with:

- React 17
- React Router 6
- Webpack 5
- TypeScript
- Jest
- Testing Library
- ESLint
- Prettier
- Husky

## Package.json

Be sure and update the `name`, `description`, and `author` fields in `package.json`.
